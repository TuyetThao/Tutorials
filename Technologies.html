The encryption algorithm is an encryption process that transforms information by mathematical methods to generate a string of cryptic characters for security purposes, all information is only authorized parties can access it and those who do not have authority. Access is not possible (proofpoint, nd). Choosing the encryption algorithm to secure customer information is what JustGo wants to ensure because JustGo wants to build a brand name that is reliable, user-friendly and extremely useful.



Encryption does not prevent information from being stolen but the stolen information cannot be used or read, encryption brings more security for information especially in the 4.0 era like today. Just like when we use ATMs or make online payments, encryption is used to secure information from harm. There are 3 main components for storing encryption systems: data, encryption engine, and key management (Aislyn, 2015).



Symmetric and asymmetric are two types of ciphers that are widely used to decide which cryptography best conceals information, and which variables are used as the key to make the encrypted message unique (Margaret, 2020).


  • The symmetric cipher is seen as secret key encryption and uses a single key. Symmetric-key algorithms are divided into stream ciphers and block ciphers. There are a lot of algorithms that are used such as Twofish, IDEA Serpent, AES, RC4, TDES, etc (Zainul, 2018).

  • Asymmetric ciphers are seen as public-key encryption, use 2 different keys. One private key for keeping secrets and 1 public key.

Chatbots is a computer program that can interact with people and make a conversation with people which is defined as AI based computer program. A chatbot is also considered as a virtual assistant when it can process requests or answer human queries quickly (Snigdha, 2020). Chatbots are based on analyzing the purpose and request of the user and extracting relevant feedback. Chatbots use 3 classification methods: pattern matching, understanding the natural language (NLU), and processing the natural language (NLP).



  • Pattern matching: The developer will prepare text circuits so that the Bot can respond appropriately to the customer and guide the customer. For example:


  • Natural language understanding (NLU): This is the process in which chatbots convert text into structured data for machine understanding. In order to respond to customers, Bot message analysis must complete the following 3 steps: entities, context, and expectations (Snigdha, 2020).

    • Entities: it is based on a specific request, keywords.

    • Context: it will rely on the flow of the conversation so that it can give a reasonable response without requiring the user to repeat their previous requests.

    • Expectation: Bot will give the most appropriate response for the current situation which can fulfill the customer satisfaction.

  • Natural language processing (NLP): There are 5 steps to processing the natural language

    • Tokenization: encodes the message into a language that the computer can understand.

    • Sentiment analysis: control emotions through customers' words to give positive feedback tailored to their feelings.

    • Normalization: Bot will check for errors in the text and filter out words that can change the meaning of the text.

    • Entity recognition: it filters out keywords/categories to constitute the content of the conversation so that it understands the content the user wants to convey.

    • Dependency Parsing: it uses the most common phrases.

GPS in real time


The Global Position System (GPS) is currently the only system that allows you to accurately position your position whether you are moving or standing still, it can still locate your position regardless of the weather and at any given moment. To give customers suggestions, we need to know where their location is, thereby guiding them the way and location through their location signal. The GPS system has 3 segments:


  • There are 28 satellites located in different orbits, traveling at 7000 miles per hour and 11,000 nautical miles from Earth (Library of Congress, n.d).

  • There are a lot of receivers to transmit signals such as phones, cars, smartwatches, laptops, computers, aircraft, cameras, ...

  • To ensure the satellites work properly, the control segment consists of ground stations that are located around the world.


We are determined based on how satellites work as stars in the sky we have known their location, the ground stations emit radio signals to detect your location through receivers. This system is capable of recording data. GPS was created to protect global security to guard border areas as well as track down bad guys who are trying to hide. At first, the military did not openly use GPS as widely as it is now because they fear that bad guys, those with a reactionary plot, will use it for bad purposes. Therefore, when GPS was announced for widespread use, some functions were restricted for global security purposes.
